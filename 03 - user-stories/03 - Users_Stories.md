## User Story: Request Appointment

As a patient,  
I want to request a medical appointment online,  
so that I can avoid making phone calls.

**Acceptance Criteria:**
- I must be able to select specialty, doctor, and date.
- I must receive an email confirmation.

---

## User Story: View My Appointments

As a patient,  
I want to view all my scheduled appointments,  
so that I know when I need to attend.

**Acceptance Criteria:**
- I must be able to see a list of upcoming and past appointments.
- I must be able to cancel an appointment if needed.

---

## User Story: Register Availability

As a doctor,  
I want to define my available days and time slots,  
so that patients can request appointments.

**Acceptance Criteria:**
- The system must allow selecting available dates and time slots.
- Only appointments within the defined availability can be requested.
